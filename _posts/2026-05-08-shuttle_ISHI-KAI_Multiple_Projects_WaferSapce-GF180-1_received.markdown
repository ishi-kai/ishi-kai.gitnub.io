## [Wafer.Sapce GF180 Run 1](https://www.crowdsupply.com/wafer-space/gf180mcu-run-1/) のベアダイとCoBが到着！！！
[Wafer.Sapce GF180 Run 1 への投稿完了！！！](https://ishi-kai.org/waferspace/shuttle/gf180/2025/12/01/shuttle_ISHI-KAI_Multiple_Projects_WaferSapce-GF180-1_submitted.html)にて、製造していたWafer.Sapce GF180 Run 1のベアダイとCoBが到着しました。  
コネクタが高密度コネクタのため、ピッチ変換ボードなどが必要なので、準備ができ次第、いろいろと試してみたいと思います。（とはいっても、標準ピン配列から変えてしまったため、GNDが繋がってはいけないところにつながっている気がしていますが・・・）

## [Wafer.Sapce](https://wafer.space/)とは？
[GoogleでOpenMPW](https://developers.google.com/silicon?hl=ja)のボスであったTimが、Googleを離れて今年4月くらいに[Wafer.Sapce](https://wafer.space/)を立ち上げました。そして、4か月くらいの準備期間を経て[GF180をベースとしたチップ作成のクラファン](https://www.crowdsupply.com/wafer-space/gf180mcu-run-1/)がスタートしました。  いまは、[Wafer.Sapce RUN2](https://www.crowdsupply.com/wafer-space/gf180mcu-run-2)を募集中ですので、チャレンジしてみたい方はぜひ応募してみてください。
