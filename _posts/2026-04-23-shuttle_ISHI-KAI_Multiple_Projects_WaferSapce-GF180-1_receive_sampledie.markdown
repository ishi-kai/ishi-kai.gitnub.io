## [Wafer.Sapce GF180 Run 1](https://www.crowdsupply.com/wafer-space/gf180mcu-run-1/) のサンプルベアダイが到着しました。
「[Wafer.Sapce GF180 Run 1 への投稿完了！！！](https://ishi-kai.org/waferspace/shuttle/gf180/2025/12/01/shuttle_ISHI-KAI_Multiple_Projects_WaferSapce-GF180-1_submitted.html)」でお伝えしたチップのベアダイのサンプルが届きました。

### [Wafer.Sapce](https://wafer.space/)とは？
[GoogleでOpenMPW](https://developers.google.com/silicon?hl=ja)のボスであったTimが、Googleを離れて2025/04に[Wafer.Sapce](https://wafer.space/)を立ち上げて、開始されたオープンソース半導体ベースの製造サービスの一つです。  
GlobalFoundriesのmixedアーキテクチャ対応の180nmベースのシャトルとなります。現在、[Wafer.Space GF180MCU Run 2](https://www.crowdsupply.com/wafer-space/gf180mcu-run-2/)を募集していますので、気になる方はチェックしてみてください！
